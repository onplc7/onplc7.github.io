Windows Server Active Directory

Contexte du projet

Mise en place d'une solution Active Directory sous Windows Server pour une société fictive nommée "Pipalap". Cette simulation avait pour objectif de reproduire l'infrastructure d'une entreprise afin de centraliser la gestion des comptes, des ressources et des politiques de sécurité.

Ce projet m'a permis de concevoir l'architecture logique du domaine, d'organiser les objets Active Directory et de mettre en place des stratégies de groupe afin d'automatiser la gestion des postes et renforcer la sécurité du système d'information.

Réalisations & Compétences

  • Installation et configuration de Windows Server.
  • Déploiement du rôle Active Directory Domain Services (AD DS).
  • Configuration du service DNS intégré au domaine.
  • Création et structuration des Unités d'Organisation (OU).
  • Gestion des comptes utilisateurs, groupes et droits d'accès.
  • Mise en place de Stratégies de Groupe (GPO) pour la sécurisation et l'administration des postes.
  • Automatisation de la création d'utilisateurs via PowerShell.
